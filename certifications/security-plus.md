# CompTIA Security+

**Earned:** December 19, 2025  
**Exam:** SY0-701

---

## What This Certification Covers

### Domain 1: General Security Concepts
- Security controls (technical, managerial, operational, physical)
- CIA Triad and security principles
- Change management and auditing
- Cryptography fundamentals

### Domain 2: Threats, Vulnerabilities, and Mitigations
- Threat actors and attack vectors
- Common vulnerabilities (injection, XSS, privilege escalation)
- MITRE ATT&CK framework
- Security techniques and mitigation strategies

### Domain 3: Security Architecture
- Network security architecture
- Cloud security concepts
- Secure application development
- Authentication and authorization

### Domain 4: Security Operations
- Security monitoring and alerting
- Incident response procedures
- Digital forensics concepts
- Log analysis and SIEM basics

### Domain 5: Security Program Management and Oversight
- Risk management and assessment
- Compliance and regulations
- Security awareness training
- Business continuity and disaster recovery

## Study Approach

**Timeline:** ~4-5 weeks

**Resources Used:**
- Professor Messer video series (free on YouTube)
- Crucial Exams practice tests
- Physical notecards for terminology and attack types
- TryHackMe labs for hands-on practice

**My Study Cycle:**
1. Watch Professor Messer videos - Take notes on key concepts
2. Take Crucial Exams practice test - Identify knowledge gaps
3. Create targeted notecards - Only for topics I missed or struggled with
4. Study notecards - Focus on weak areas until consistent
5. Retake practice tests - Repeat until scoring 90%+
6. Night-before refresh - Review notecards one final time

**Key Focus:** Security+ is much broader than A+. Focus on understanding security concepts and decision-making, not just memorizing facts.

## Tips for Future Test-Takers

1. **Know your attack types** - Phishing, ransomware, SQL injection, XSS, etc.
2. **Understand security controls** - Technical, managerial, operational, physical
3. **Practice scenario questions** - Security+ loves "What should you do next?" questions
4. **Use Crucial Exams heavily** - The practice test format matches the real exam
5. **Don't just memorize port numbers** - Understand what services do and why they're targeted
6. **Focus on the "why"** - CompTIA wants you to justify security decisions
