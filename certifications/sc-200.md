# SC-200: Microsoft Security Operations Analyst

**Status:** In Progress  
**Target Completion:** Q2 2026  
**Exam:** SC-200

---

## What This Certification Covers

### Mitigate Threats Using Microsoft Defender
- Microsoft Defender for Endpoint
- Microsoft Defender for Office 365
- Microsoft Defender for Identity
- Microsoft Defender for Cloud Apps
- Threat investigation and response

### Mitigate Threats Using Microsoft Sentinel
- Data connectors and log ingestion
- KQL (Kusto Query Language) for threat hunting
- Analytics rules and automated incident response
- Workbooks and dashboards
- SOAR (Security Orchestration, Automation, and Response)

### Query, Visualize, and Monitor Data
- Advanced KQL queries
- Log analysis and correlation
- Time-series analysis
- Custom workbooks and visualizations

### Configure and Manage Automation
- Automation rules and playbooks
- Logic Apps for SOAR workflows
- Incident management automation
